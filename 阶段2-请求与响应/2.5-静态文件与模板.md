# 2.5 静态文件与模板

> FastAPI 不仅是纯 API 框架——它也能托管静态资源和渲染服务端页面。虽然大多数项目前后端分离，但掌握这些能力能让你应对各种场景。

---

## 1. 为什么需要静态文件和模板？

你可能会想："我是做后端 API 的，为什么还要学 HTML 模板和静态文件？" 实际上，就算是在纯 API 项目中，你也很可能遇到以下场景：

| 场景 | 说明 |
|------|------|
| **文件访问** | 用户上传的头像、图片需要公开访问 URL——这需要 StaticFiles 挂载 |
| **后台管理** | 给运营团队用的数据后台，不需要单独开发一套前端项目 |
| **邮件模板** | 注册确认邮件、通知邮件的 HTML 内容——用 Jinja2 渲染比硬编码方便得多 |
| **快速原型** | 给老板或客户演示时，一个能看的页面比 JSON 接口更有说服力 |
| **API 文档增强** | 在 `/docs` 旁边挂载自定义的帮助页面 |

> 简单来说：**学习本章不是为了取代前后端分离，而是为了让你在需要时能快速响应，不用每次都求前端帮忙。**

---

## 2. `StaticFiles` — 挂载静态资源

### 本质理解

`app.mount()` 做的事情很简单：它把文件系统中的一个目录"映射"成 URL 路径。访问 `http://你的域名/static/logo.png` 就等于读取 `./static/logo.png` 这个文件。

这和 Nginx 的静态文件服务是一个道理——只是 FastAPI 让你在开发阶段不用再配一个 Nginx。

---

## 2. `StaticFiles` — 挂载静态资源

### 2.1 基础用法

```python
from fastapi import FastAPI
from fastapi.staticfiles import StaticFiles

app = FastAPI()

# 挂载 /static 路径 → ./static 目录
app.mount("/static", StaticFiles(directory="static"), name="static")
```

目录结构：

```
project/
├── main.py
└── static/
    ├── images/
    │   └── logo.png          → GET /static/images/logo.png
    ├── css/
    │   └── style.css         → GET /static/css/style.css
    └── js/
        └── app.js            → GET /static/js/app.js
```

### 2.2 完整配置

```python
from fastapi import FastAPI
from fastapi.staticfiles import StaticFiles

app = FastAPI()


# 基础：映射整个目录
app.mount(
    "/static",
    StaticFiles(directory="static"),
    name="static",
)

# 高级：更多选项
app.mount(
    "/assets",
    StaticFiles(
        directory="assets",
        html=True,             # 自动提供目录内的 index.html
        check_dir=True,        # 启动时检查目录是否存在
        follow_symlinks=False, # 是否跟踪符号链接
    ),
    name="assets",
)

# 文件上传目录
app.mount(
    "/uploads",
    StaticFiles(directory="uploads"),
    name="uploads",
)
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `directory` | 要挂载的本地目录路径 | 必填 |
| `html` | 是否自动提供 `index.html` | `False` |
| `check_dir` | 启动时检查目录是否存在 | `True` |
| `follow_symlinks` | 是否跟踪符号链接 | `False` |
| `packages` | 从 Python 包中挂载静态文件 | `None` |

### 2.3 挂载 Python 包中的静态文件

```python
# 挂载第三方库中的静态资源
app.mount(
    "/admin-static",
    StaticFiles(
        packages=[("sqladmin", "statics")],
    ),
    name="admin-static",
)
# GET /admin-static/main.css → sqladmin 包的 statics/main.css
```

### 2.4 多个静态目录

```python
# 挂载多个目录
app.mount("/uploads", StaticFiles(directory="uploads"), name="uploads")
app.mount("/static", StaticFiles(directory="static"), name="static")
app.mount("/favicon.ico", StaticFiles(directory="static/icons"), name="icons")
```

## 3. 路由与静态文件的优先级

一个很容易踩的坑：当你同时有路由和静态文件挂载时，谁先"接住"请求？

**规则：路由优先。** FastAPI 先检查是否有路由匹配，找不到匹配的路由才把请求交给静态文件系统。

```python
app = FastAPI()

# 即使你先挂载了静态文件...
app.mount("/static", StaticFiles(directory="static"), name="static")

# ...路由依然优先！
@app.get("/static/custom.json")
async def custom_json():
    """这个路由会拦截 /static/custom.json，磁盘上的 custom.json 不会被返回"""
    return {"dynamic": True}

# 访问 /static/custom.json → 返回 {"dynamic": True}
# 访问 /static/logo.png     → 返回 static/logo.png 文件（没有匹配的路由）
```

> **实际意义**：你可以在某些特定路径上用路由动态生成内容，其余交给静态文件。比如 `/static/generated/chart.png` 由路由动态生成，`/static/logo.png` 从磁盘读取。

## 4. Jinja2 模板引擎

### 什么是模板引擎？为什么要用它？

假设你需要发一封注册确认邮件。邮件的 HTML 内容是：

```
<h1>欢迎{用户名}！</h1>
<p>你已于 {注册时间} 注册成功。</p>
```

如果不使用模板，你可能会这样写：

```python
html = f"<h1>欢迎{username}！</h1><p>你已于 {created_at} 注册成功。</p>"
```

这在小规模时没问题，但当邮件内容变得复杂（几十行 HTML、带循环和条件判断），拼字符串就变成了噩梦。模板引擎解决的问题就是**把 HTML 结构和数据分开**——HTML 放在 `.html` 文件里，数据由 Python 传入，渲染时自动合并。

### 4.1 安装与配置

```bash
pip install jinja2 aiofiles
```

```python
from fastapi import FastAPI, Request
from fastapi.templating import Jinja2Templates

app = FastAPI()

# templates = Jinja2Templates(directory="templates")
#         ↑                          ↑
#    Jinja2 模板引擎实例         模板文件存放的目录
templates = Jinja2Templates(directory="templates")
```

### 4.2 第一个模板页面：从 Python 到 HTML

模板渲染的核心流程只有三步：
1. 创建一个 HTML 模板文件，用 `{{ 变量 }}` 占位
2. 在路由函数中准备好数据（通常是字典）
3. 调用 `templates.TemplateResponse()` 把数据和模板合并输出

**Python 代码：**

```python
from fastapi import FastAPI, Request
from fastapi.templating import Jinja2Templates

app = FastAPI()

templates = Jinja2Templates(directory="templates")


@app.get("/")
async def home(request: Request):
    """渲染首页"""
    return templates.TemplateResponse(
        request=request,     # ← 必须传入 request 对象
        name="index.html",   # ← 模板文件名
        context={            # ← 传给模板的变量
            "title": "FastAPI 学习项目",
            "user": {
                "name": "小明",
                "role": "学习者",
            },
            "items": [
                {"id": 1, "title": "学 FastAPI 核心", "done": True},
                {"id": 2, "title": "学依赖注入", "done": False},
                {"id": 3, "title": "学数据库", "done": False},
            ],
        },
    )
```

**`templates/index.html`：**

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>{{ title }}</title>
    <style>
        body { font-family: system-ui, sans-serif; max-width: 800px; margin: 0 auto; padding: 2rem; }
        .done { text-decoration: line-through; color: #888; }
    </style>
</head>
<body>
    <h1>{{ title }}</h1>
    <p>欢迎，{{ user.name }}（{{ user.role }}）！</p>

    <h2>学习进度</h2>
    <ul>
        {% for item in items %}
            <li class="{{ 'done' if item.done else '' }}">
                {{ item.id }}. {{ item.title }}
                {% if item.done %}✅{% else %}⬜{% endif %}
            </li>
        {% endfor %}
    </ul>
</body>
</html>
```

### 4.3 Jinja2 模板语法速查

下面是最常用的语法，掌握了这些就能应对 90% 的模板渲染场景。不需要背，用到时回来查就行。

```jinja2
{# ═══════════ 输出变量 ═══════════ #}
{{ variable }}
{{ user.name }}          {# 访问属性 #}
{{ items[0] }}           {# 访问列表元素 #}

{# ═══════════ 过滤器（对变量做转换）═══════════ #}
{{ name | upper }}       {# 转大写 #}
{{ name | default("匿名") }}  {# 变量为空时用默认值 #}
{{ price | round(2) }}       {# 保留两位小数 #}
{{ text | truncate(100) }}   {# 截断到 100 个字符 #}

{# ═══════════ 条件判断 ═══════════ #}
{% if user.role == "admin" %}
    <a href="/admin">管理后台</a>
{% elif user.role == "editor" %}
    <a href="/editor">编辑面板</a>
{% else %}
    <p>普通用户</p>
{% endif %}

{# ═══════════ 循环 ═══════════ #}
{% for item in items %}
    <li>
        {{ loop.index }}.      {# 序号，从 1 开始 #}
        {{ item.title }}
        {% if loop.first %} 首个 {% endif %}
        {% if loop.last %}  最后一个 {% endif %}
    </li>
{% else %}
    <li>列表为空</li>           {# 当 items 为空时显示这一段 #}
{% endfor %}

{# ═══════════ 模板继承 ═══════════ #}
{% extends "base.html" %}       {# 继承基础布局 #}
{% block content %}             {# 填充父模板定义的 content 块 #}
    <p>这是页面内容</p>
{% endblock %}

{# ═══════════ 包含子模板 ═══════════ #}
{% include "header.html" %}     {# 直接把一段 HTML 嵌入到这里 #}

{# ═══════════ 模板注释（不会出现在输出的 HTML 中）═══════════ #}
{# 这是注释，浏览器看不到 #}
<!-- 这是 HTML 注释，浏览器能看到 -->
```

## 5. 后台管理页面实战

### 5.1 API 端点

```python
"""
简易后台管理 — 结合 API、静态文件、模板
"""
from fastapi import FastAPI, Request, HTTPException, Form
from fastapi.staticfiles import StaticFiles
from fastapi.templating import Jinja2Templates
from fastapi.responses import RedirectResponse
from datetime import datetime

app = FastAPI(title="简易后台管理")

# ── 挂载静态文件 ──
app.mount("/static", StaticFiles(directory="static"), name="static")

# ── 模板 ──
templates = Jinja2Templates(directory="templates")

# ── 模拟数据库 ──
items_db: dict[int, dict] = {
    1: {"id": 1, "name": "Python FastAPI 入门", "price": 49.9, "stock": 100, "created_at": "2024-01-15"},
    2: {"id": 2, "name": "Vue.js 实战", "price": 59.9, "stock": 50, "created_at": "2024-02-20"},
    3: {"id": 3, "name": "Docker DevOps", "price": 79.9, "stock": 30, "created_at": "2024-03-10"},
}
_next_id = 4

# ── 辅助函数 ──
def get_stats() -> dict:
    """计算统计数据"""
    total_items = len(items_db)
    total_stock = sum(item["stock"] for item in items_db.values())
    total_value = sum(item["price"] * item["stock"] for item in items_db.values())
    return {
        "total_items": total_items,
        "total_stock": total_stock,
        "total_value": round(total_value, 2),
    }

# ── 页面路由 ──
@app.get("/admin")
async def admin_dashboard(request: Request):
    """后台首页 - 仪表盘"""
    stats = get_stats()
    return templates.TemplateResponse(
        "admin/dashboard.html",
        {
            "request": request,
            "title": "管理后台",
            "stats": stats,
            "items": list(items_db.values()),
        },
    )


@app.get("/admin/items/new")
async def admin_new_item_form(request: Request):
    """新建商品表单页"""
    return templates.TemplateResponse(
        "admin/item_form.html",
        {
            "request": request,
            "title": "新建商品",
            "item": None,  # 新建模式
        },
    )


@app.get("/admin/items/{item_id}/edit")
async def admin_edit_item_form(request: Request, item_id: int):
    """编辑商品表单页"""
    if item_id not in items_db:
        raise HTTPException(status_code=404, detail="商品不存在")
    return templates.TemplateResponse(
        "admin/item_form.html",
        {
            "request": request,
            "title": f"编辑：{items_db[item_id]['name']}",
            "item": items_db[item_id],
        },
    )


# ── API 端点（被表单提交调用）──
@app.post("/admin/items")
async def admin_create_item(
    name: str = Form(...),
    price: float = Form(..., gt=0),
    stock: int = Form(..., ge=0),
):
    """创建商品（表单提交）"""
    global _next_id
    items_db[_next_id] = {
        "id": _next_id,
        "name": name,
        "price": price,
        "stock": stock,
        "created_at": datetime.now().strftime("%Y-%m-%d"),
    }
    _next_id += 1
    # POST 后重定向到列表页（PRG 模式）
    return RedirectResponse(url="/admin", status_code=303)


@app.post("/admin/items/{item_id}/update")
async def admin_update_item(
    item_id: int,
    name: str = Form(...),
    price: float = Form(..., gt=0),
    stock: int = Form(..., ge=0),
):
    """更新商品（表单提交）"""
    if item_id not in items_db:
        raise HTTPException(status_code=404, detail="商品不存在")
    items_db[item_id].update({
        "name": name,
        "price": price,
        "stock": stock,
    })
    return RedirectResponse(url="/admin", status_code=303)


@app.get("/admin/items/{item_id}/delete")
async def admin_delete_item(item_id: int):
    """删除商品"""
    if item_id not in items_db:
        raise HTTPException(status_code=404, detail="商品不存在")
    del items_db[item_id]
    return RedirectResponse(url="/admin", status_code=303)
```

## 6. 重要注意事项

### 6.1 生产环境：静态文件交给 Nginx

FastAPI 的 `StaticFiles` 在开发环境很方便，但它本质上是让 Python 进程去读磁盘文件——性能远不如专门的 Web 服务器。生产环境的标准做法是：

```
Nginx 处理静态文件 → FastAPI 只处理 API 请求
```

```python
# 开发环境：FastAPI 自己处理一切
app.mount("/static", StaticFiles(directory="static"), name="static")
app.mount("/uploads", StaticFiles(directory="uploads"), name="uploads")

# 生产环境：Nginx 拦截 /static/ 和 /uploads/，
# 直接返回文件，请求根本不会到达 FastAPI

# Nginx 配置片段（了解即可，阶段 8 会详细讲）：
# location /static/ {
#     alias /var/www/myapp/static/;
#     expires 30d;              # 浏览器缓存 30 天
#     add_header Cache-Control "public, immutable";
# }
# location /uploads/ {
#     alias /var/www/myapp/uploads/;
# }
```

> 如果你还没有 Nginx，开发阶段直接用 `StaticFiles` 就行——不要过早优化。

### 6.2 缓存控制

```python
from fastapi import FastAPI
from fastapi.staticfiles import StaticFiles

app = FastAPI()

# 自定义响应头
class CachedStaticFiles(StaticFiles):
    async def get_response(self, path: str, scope):
        response = await super().get_response(path, scope)
        # 静态资源设置长期缓存
        if path.endswith((".js", ".css", ".png", ".jpg", ".svg", ".woff2")):
            response.headers["Cache-Control"] = "public, max-age=31536000, immutable"
        return response

app.mount("/static", CachedStaticFiles(directory="static"), name="static")
```

### 6.3 模板中的 `url_for`

```python
from fastapi import FastAPI, Request
from fastapi.templating import Jinja2Templates

app = FastAPI()

# 给静态文件挂载命名
app.mount("/static", StaticFiles(directory="static"), name="static")

templates = Jinja2Templates(directory="templates")

# Jinja2 中可以使用 url_for
# {{ url_for('static', path='/css/style.css') }}
# → /static/css/style.css
```

在模板中使用：

```html
<link rel="stylesheet" href="{{ url_for('static', path='/css/style.css') }}">
<script src="{{ url_for('static', path='/js/app.js') }}"></script>
<img src="{{ url_for('static', path='/images/logo.png') }}" alt="Logo">
```

### 6.4 `html=True` 的陷阱

```python
# ⚠️ 当 html=True 时，访问 /static/ 会返回 index.html
app.mount("/static", StaticFiles(directory="static", html=True), name="static")

# 如果 static/ 目录下没有 index.html，访问 /static/ 返回 404
# 如果不想要这个行为，设 html=False
```

## 7. 综合实战：小型 CMS 后台

```python
"""
综合实战 — 小型 CMS 后台
功能：文章管理（列表 / 创建 / 编辑 / 删除）

项目结构：
static/
    admin.css
templates/
    admin/
        base.html        # 后台布局
        dashboard.html   # 仪表盘
        article_form.html # 文章表单
"""
from fastapi import FastAPI, Request, Form, HTTPException
from fastapi.staticfiles import StaticFiles
from fastapi.templating import Jinja2Templates
from fastapi.responses import RedirectResponse
from datetime import datetime

app = FastAPI(title="小型 CMS")

app.mount("/static", StaticFiles(directory="static"), name="static")
templates = Jinja2Templates(directory="templates")

# 模拟文章数据库
articles_db: dict[int, dict] = {
    1: {
        "id": 1, "title": "FastAPI 入门指南", "content": "FastAPI 是一个现代...",
        "status": "published", "author": "小明",
        "created_at": "2024-01-15", "updated_at": "2024-01-15",
    },
    2: {
        "id": 2, "title": "Python 异步编程", "content": "asyncio 是 Python...",
        "status": "draft", "author": "小明",
        "created_at": "2024-02-20", "updated_at": "2024-02-20",
    },
}
_next_id = 3


@app.get("/admin")
async def dashboard(request: Request):
    """管理后台首页"""
    drafts = [a for a in articles_db.values() if a["status"] == "draft"]
    published = [a for a in articles_db.values() if a["status"] == "published"]
    return templates.TemplateResponse("admin/dashboard.html", {
        "request": request,
        "drafts": drafts,
        "published": published,
        "total": len(articles_db),
    })


@app.get("/admin/articles/new")
async def new_article_form(request: Request):
    """新建文章"""
    return templates.TemplateResponse("admin/article_form.html", {
        "request": request,
        "article": None,
    })


@app.get("/admin/articles/{article_id}/edit")
async def edit_article_form(request: Request, article_id: int):
    """编辑文章"""
    article = articles_db.get(article_id)
    if not article:
        raise HTTPException(status_code=404, detail="文章不存在")
    return templates.TemplateResponse("admin/article_form.html", {
        "request": request,
        "article": article,
    })


@app.post("/admin/articles")
async def create_article(
    title: str = Form(..., min_length=1),
    content: str = Form(..., min_length=1),
    author: str = Form(..., min_length=1),
    status: str = Form(default="draft"),
):
    """创建文章"""
    global _next_id
    now = datetime.now().strftime("%Y-%m-%d %H:%M")
    articles_db[_next_id] = {
        "id": _next_id,
        "title": title,
        "content": content,
        "author": author,
        "status": status,
        "created_at": now,
        "updated_at": now,
    }
    _next_id += 1
    return RedirectResponse(url="/admin", status_code=303)


@app.post("/admin/articles/{article_id}/update")
async def update_article(
    article_id: int,
    title: str = Form(..., min_length=1),
    content: str = Form(..., min_length=1),
    author: str = Form(..., min_length=1),
    status: str = Form(default="draft"),
):
    """更新文章"""
    article = articles_db.get(article_id)
    if not article:
        raise HTTPException(status_code=404, detail="文章不存在")
    article.update({
        "title": title,
        "content": content,
        "author": author,
        "status": status,
        "updated_at": datetime.now().strftime("%Y-%m-%d %H:%M"),
    })
    return RedirectResponse(url="/admin", status_code=303)


@app.get("/admin/articles/{article_id}/delete")
async def delete_article(article_id: int):
    """删除文章"""
    if article_id not in articles_db:
        raise HTTPException(status_code=404, detail="文章不存在")
    del articles_db[article_id]
    return RedirectResponse(url="/admin", status_code=303)
```
